Code Buddy

Code Buddy is a simple desktop application designed for conducting coding assignments!

Features of Code Buddy

  • No registration required
  • Support for multiple languages i.e. C, C++ and Java
  • Create problem once and share with your students via a code-buddy(*.cb) file
  • Correction against multiple test cases and a quick result

Dependency

You need to first install JDK(Java Development Kit) or JRE(Java Runtime Environment) version - 8 in order to run this application
You can download the JDK8 from here
For those having Linux OS can install JDK8 via quick command as following:
sudo apt-get update
sudo apt-get install openjdk-8-jdk
A compiler of your choice i.e. gcc-compiler for C, g++ compiler for C++ and JDK8 you already have for Java
Don't forget to configure your path variables!

Download and Installation

Click here to download the JAR file
Once you have installed JDK8 and configured the path variables then you can run the application in either way:
  1. Double click on the application
  2. Run the command java -jar CodeBuddy.jar
